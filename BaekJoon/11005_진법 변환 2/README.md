## 백준 2745번: 진법 변환

- [백준 1152번: 단어의 개수](https://www.acmicpc.net/problem/2745)

---

### 📖 설명

- 10진수 N이 주어진다.
- 이 수를 B진법으로 바꿔 출력하는 프로그램을 작성하시오.

---

**예시:**

입력:

```
60466175 36
```

출력:

```
ZZZZZ
```

---

### 📝 입력

- 첫째 줄에 N과 B가 주어진다. (2 ≤ B ≤ 36)
- 10진수 N을 B진법으로 바꾸면, 항상 10억보다 작거나 같다.

### 📤 출력

- 첫째 줄에 10진수 N을 B진법으로 출력한다.
